Module: core
Parent Module: rabbit-hole

The terrain system.

Constructor

Terrain

(
  • [options]
)

Defined in src\core\terrain.js:51

Parameters:

  • [options] Object optional
    • The options.
    • [chunkSize=32] Number optional
      • The world size of a volume chunk. Will be rounded up to the next power of two.
    • [resolution=32] Number optional
      • The resolution of a volume chunk. Will be rounded up to the next power of two.
    • [maxWorkers] Number optional
      • Limits the amount of active workers. The default limit is the amount of logical processors which is also the maximum.
    • [levels] Number optional
      • The amount of detail levels. The default number of levels is derived from the resolution.
